Требования к JSON файлам:
  • Должен содержать валидный JSON с метаданными фотографий
  • Обязательные поля: filename, format, file_size, width, height
  • Файлы автоматически проверяются на валидность
  • Невалидные файлы автоматически удаляются
Пример структуры JSON:
{
  "filename": "photo1.jpg",
  "format": "JPEG",
  "file_size": 2048576,
  "width": 1920,
  "height": 1080,
  "camera_make": "Canon",
  "camera_model": "EOS 5D Mark IV"
}
